pyspread.src.lib.exception_handling module¶

Exception Handeling¶

exception_handling.py contains functions for handeling exceptions generated by

user code

Provides¶

  • get_user_codeframe: Returns traceback that only includes the user’s execute

    code frames

pyspread.src.lib.exception_handling.get_user_codeframe(tb)[source]¶

Modify traceback to only include the user code’s execution frame Always call in this fashion:

e = sys.exc_info() user_tb = get_user_codeframe(e[2]) or e[2]

so that you can get the original frame back if you need to (this is necessary because copying traceback objects is tricky and this is a good workaround)
