Code base

This information refers to pyspread v. 0.2.3. The image on the right refers to v. 0.2.2. Therefore, it is missing the module chart_dialog.py, which implements the chart editing dialog.

Version 0.1.2 and previous have a different structure.

The code is organized in the src folder. There are the following sub-folders:

  • actions: Actions that change the model
  • gui: GUI elements including events
  • model: Core data structure
  • lib: Generically usable libraries

The figure on the right displays class instance dependencies of the different layers.

Relations to dependencies

wxPython related code can be found

  • in the action folder files,

  • in the gui folder files (including events.py),

  • in lib/clipboard.py,

  • in lib/gpg.py,

  • in lib/parsers.py,

  • in config.py,

  • in pyspread.py,

  • in sysvars.py and

  • in model/model.py.

The latter is necessary in order to use the wx constants in config.py


python-gnupg related code can be found in lib/gpg.py


numpy related code can be found in model/model.py
