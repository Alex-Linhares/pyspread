How to help

Thank you for wanting to help pyspread. Bug reports, translations, icons, deployment to further operation systems, code, ideas etc. are most welcome.

For bug reports, please follow the link in the menu to the pyspread issue tracker on github. For anything else please send an e-mail to pyspread-dev@gna.org.

Localization

Pyspread is localized via the Translation project, i.e. the textual content of the user interface is translated into different languages. If you like to localize pyspread, please refer to the translators' page for detailed instructions.

Localization status (for latest additions see the translation project page):

Submitting code

  1. Get a github account.
  2. Fork the branch that you would like to work with (see github help).
  3. Issue a pull request when ready.

or send your patch to pyspread-dev@gna.org
